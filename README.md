### System

typescript 3.8.3
webpack 4.42.0

### Development server
~~~
ng serve
~~~

### Express server
~~~
node server.js
~~~

### Code scaffolding
~~~
ng generate component component-name

~~~

### Build
~~~
ng build
ng build --prod
~~~

### Running tests
~~~
ng test
ng e2e
~~~

### Upgrade package-lock.json
~~~
npm update
npm audit fix

yarn upgrade
~~~
